Markdown语法总结

Markdown语法总结

1.标题

添加标题的语法为：

#这是一级标题的写法

##这是二级标题的写法

###这是三级标题的写法

####这是四级标题的写法

#####这是五级标题的写法

######这是六级标题的写法

示例如下：

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

2.插入列表

列表的语法为：

- Web安全

    - 跨站脚本攻击

        - 定义

        - 危害

        - 先决条件

        - 流程

    - SQL注入

    - Google hacking

    ……

即’- ‘ （注意’-‘之后一定有空格）

注：’-‘也可替换成’*’

示例如下：

  • Web安全
    • 跨站脚本攻击
      • 定义
      • 危害
      • 先决条件
      • 流程
    • SQL注入
    • Google Hacking

注：使用’- ‘,’+ ‘可实现有序列表

3.插入超链接

插入超链接的语法为：

[这是一条超链接，点击即可访问百度](www.baidu.com)

即用[]括起要显示的文字，并在()内写出对应的URL

示例如下：

点我即可访问百度

4.插入图片

插入图片的方式和插入超链接的方式类似，相应语法为

![文字](URL)

示例如下：

5.引用

添加引用的语法为：

>这是一条引用的语句

即使用’>’ 然后接引用的文字

示例如下：

这是一条引用

6.斜体，粗体，粗斜体

斜体的语法为：

*这是斜体*

粗体的语法为：

**这是粗体**

粗斜体的语法为：

***这是粗斜体***

示例如下：

这是斜体

这是粗体

这是粗斜体

7.插入代码

插入代码的语法为：

\

即用’

int main(void){

printf("Hello World");

return 0;

}

`
